Arrays

.each Method

        arr.each { |item| statements }
Use the each method to iterate through the array, temporarily storing the value in the item placeholder.

.each_index Method

        arr.each_index { |index| statements }
Use the each_index method to iterate through the array, temporarily storing the index number in the index placeholder, NOT the value from the array.

.collect and .collect! Methods

        arr.collect { |item| statements }
While the collect method iterates, it returns the value from the block statements, creating an entirely new array, or replacing the array if using the destructive collect! method.

.reverse_each Method

        arr.reverse_each { |item| statements }
The reverse_each method is similar the each method, but works backwards through the array instead.

Hashes

.each Method

        hash.each { |key, value| statements }
Use the each method to iterate through the hash, temporarily storing both the current key and value in variables.

.each_key Method

        hash.each { |key| statements }
Use the each_key method to iterate through the hash, temporarily storing only the current key for use.

.each_value Method

        hash.each_value { |value| statements }
Use the each_value method to iterate through the hash, temporarily storing only the current value for use.
